Alors, la librairie Cairo m'a l'air d'être pas mal pour essayer de faire les visuels des noeuds

J'ai galéré à trouver comment setup (surtout où le télécharger à vrai dire), mais dcp pour résumer :

1- Installer MSYS2 sur https://www.msys2.org (dans C:, sinon les chemins marchent plus)
2- Ouvrir le terminal MSYS2 UCRT64, et faire les update avec
    pacman Syu (ça va relancer) puis
    pacman Su 
3- Toujours dans ce terminal, 
    pacman -S mingw-w64-ucrt-x86_64-gcc mingw-w64-ucrt-x86_64-cairo 
4- Ajouter C:\msys64\ucrt64\bin au PATH du système windows
5- Ajouter un terminal UCRT64 à VSCode dans "terminal.integrated.profiles.windows" (et l'utiliser dcp)
    "terminal.integrated.profiles.windows": {
        "MSYS2 UCRT64": {
        "path": "C:\\msys64\\usr\\bin\\bash.exe",
        "args": ["--login", "-i"],
        "env": { "MSYSTEM": "UCRT64" }
        }
    }
6- Pour compiler: gcc graphicstest.c -IC:/msys64/ucrt64/include -LC:/msys64/ucrt64/lib -lcairo -o graphicstest.exe

et normalement tout devrait marcher avec les fichiers json
